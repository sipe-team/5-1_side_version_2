#### 준비 내용
- 팀별
  - 미션동기
  - 하고자했던 방향과 작업
  - 과정과 결과물 소개
  - 미션을 통해서 얻고자 한 것(얻은 것)
  - 작업 내용 소개(구체적인 내용)
  - 한 줄 느낌점
- 개인
  - 1-2줄 소감

- 발표 내용 정리 데드라인 -> 목요일 (23시 59분)
  - 발표 내용에 포함하고 싶은 것들이 있다면 그 전에 PR 도 올려주시고, 내용 정리를 마무리 해서 그 부분은 목요일 전으로 끝내는 것으로
  - Docs 에 정리해서 PR 올리기
  - 혹시나 지각이나 제출하지 않은 것들을 대비해서. 벌금 부과
  - 미제출 : 10만원
  - 늦게 제출 : 2만원

- 발표자
  - 각 팀별 한 명씩
    - 컴포넌트 팀 : 원주님
    - 문서화 팀 : 진호님
  - 발표 컨텐츠 및 내용 정리는 같이 얘기 후 준비
  - 팀별로 따로 미팅을 가지거나 발표 관련 논의
