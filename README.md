## 구조

- `members/` : 참여자별 활동 기록
- `weeks/` : 주차별 목표 및 회의 메모
- `discussions/` : 의사결정 기록 및 action item 정리
- `references/` : 참고 자료 정리


## 운영 방식
- 각 참여자는 자기 브랜치에서 작업 후 PR 생성
- 주차별 회의 이후 내용 반영 및 머지
- 관련 PR 링크를 각 문서에 기록
